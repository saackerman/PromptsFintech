## How to Use This Repository

### Quick Start (5 minutes)
1. Choose a category
2. Select a prompt
3. Test with 2-3 AI systems
4. Compare responses
5. Record results

### Deep Dive (30-60 minutes)
1. Select 3-5 prompts from different categories
2. Test with multiple AI systems
3. Use evaluation framework
4. Score responses (0-50 points)
5. Document findings
